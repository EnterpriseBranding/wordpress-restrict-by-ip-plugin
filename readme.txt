Restrict access to site(s), pages and/or content access by IP address or subnet mask.

== Description ==
This plugin allows you to restrict access to an entire site, network sub-sites,
individual pages and content by shortcode. Multisite supported.

= Requirements =

* WordPress 4.0 or higher
* PHP 5.6 or higher
* [Carbon Fields](https://carbonfields.net/release-archive/) 2.2.0 or higher

== Installation ==
1. Unzip the downloaded distributable zip file
2. Upload the extracted folder and its contents into the `wp-content/plugins/` directory of your WordPress installation
3. Activate *Restrict by IP* from Plugins page

== Frequently Asked Questions ==

= Can I restrict by a pattern/subnet mask? =

Yes. Regular expressions are supported for restricting by patterns.
